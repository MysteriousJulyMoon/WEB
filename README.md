# PHP Learning Projects

Коллекция небольших PHP-проектов и упражнений, демонстрирующих различные аспекты программирования на PHP: работу с массивами, взаимодействие с базами данных, веб-формы и базовые алгоритмы.

## 📁 Структура проекта
php-learning/
├── array-exercises/ # Упражнения по работе с массивами

│ ├── array_operations.php # Базовые операции с массивами

│ ├── multidimensional.php # Многомерные массивы

│ └── functions.php # Работа с array_ функциями

├── web-applications/ # Веб-приложения

│ ├── advertisement-board/ # Доска объявлений

│ │ ├── index.php # Главная страница с формой

│ │ ├── config.php # Конфигурация БД

│ │ └── database/ # SQL-скрипты для БД

│ ├── calculator/ # Калькулятор

│ └── contact-form/ # Контактная форма

├── database-examples/ # Примеры работы с БД

│ ├── mysqli-example.php # Работа с MySQLi

│ └── pdo-example.php # Работа с PDO

├── algorithms/ # Алгоритмические задачи

│ ├── sorting.php # Алгоритмы сортировки

│ ├── search.php # Алгоритмы поиска

│ └── math-functions.php # Математические функции

└── utilities/ # Вспомогательные скрипты

├── file-upload.php # Загрузка файлов

├── session-example.php # Работа с сессиями

└── cookie-example.php # Работа с cookies
